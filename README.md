# Transformers

## Uruchomienie
Wymagana wersja pythona >= 3.10, ja robiłem na 3.14
- `git clone https://github.com/MaciejKuzmicki/Transformers`
- `cd Transformers`
- `python3.14 -m venv .transformers`
- `.\.transformers\Scripts\activate`
- `pip install -r requirements.txt`

Za kolejnym razem:
- w katalogu z projektem `git pull`
- `.\.transformers\Scripts\activate`
- `pip install -r requirements.txt`
- możemy już pracować z notebook'ami


## Praca z repo
- po dodaniu nowych bibliotek warto przed wrzuceniem zmian zrobić
  `pip freeze > requirements.txt`, aby zaktualizować środowisko
